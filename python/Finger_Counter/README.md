# Real-time Finger Counter  
**شمارنده تعداد انگشت‌های باز شده با وبکم (بدون نیاز به دستکش یا سنسور)**

![demo](screen-shot.png) 



### توضیح پروژه
این برنامه با استفاده از دوربین وبکم و مدل MediaPipe گوگل، در لحظه تشخیص می‌ده چند تا از انگشت‌های دست باز هستن (از ۰ تا ۵) و عدد رو روی تصویر نمایش می‌ده.

- انگشت شست: با مقایسه موقعیت افقی نوک شست (نقطه ۴) با نقطه ۳  
- چهار انگشت دیگر: با مقایسه موقعیت عمودی نوک انگشت (۸،۱۲،۱۶،۲۰) با نقطه زیرش (۷،۱۱،۱۵،۱۹)

### تکنولوژی‌ها
- **OpenCV** → خواندن و نمایش فریم‌های وبکم
- **Google MediaPipe Hands** → تشخیص ۲۱ نقطه کلیدی دست با دقت بالا و سرعت عالی
- **Python** → منطق تشخیص انگشت باز/بسته

عدد بزرگ قرمز گوشه تصویر تعداد انگشت‌های باز رو نشون می‌ده
برای خروج کلید q رو بزن
