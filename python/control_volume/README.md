# Hand Gesture Volume Control  
**کنترل صدای ویندوز با فاصله بین انگشت شست و اشاره**
### پیش نمایش
![demo](output.gif)
### توضیح پروژه
این برنامه با استفاده از دوربین وبکم، فاصله بین نوک انگشت شست و اشاره رو تشخیص می‌ده و به صورت لحظه‌ای صدای سیستم (ویندوز) رو کنترل می‌کنه:

- هر چی انگشت‌ها به هم نزدیک‌تر → صدا کمتر  
- هر چی دورتر → صدا بیشتر  
- محدوده: از بی‌صدا (-65 dB) تا حداکثر (0 dB)

### تکنولوژی‌های استفاده شده
- **OpenCV** → خواندن تصویر از وبکم و نمایش
- **Google MediaPipe** → تشخیص دقیق نقاط دست (Hand Landmarks)
- **pycaw** → کنترل مستقیم صدای ویندوز (بدون نیاز به ماوس/کیبورد)
- **NumPy** → محاسبات ریاضی

### نصب و اجرا (خیلی ساده!)

```bash
# 1. کلون کردن پروژه
git clone https://github.com/shabanvahid0101/top_projects.git
cd top_projects/python/control_volume

# 2. ساخت محیط مجازی (اختیاری اما توصیه می‌شود)
python -m venv venv
venv\Scripts\activate  # ویندوز
# source venv/bin/activate  # مک/لینوکس

# 3. نصب کتابخانه‌ها
pip install opencv-python mediapipe numpy pycaw comtypes
or
pip install -r requirements.txt
# 4. اجرای برنامه
python volume.py
```
